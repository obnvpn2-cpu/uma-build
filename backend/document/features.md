# UmaBuild 特徴量カタログ

UmaBuild が提供する特徴量の一覧です。
10カテゴリ・全 **81特徴量**、うち **34個** がデフォルトで ON になっています。

> このドキュメントは `backend/services/feature_catalog.py` から生成しています。ソースを変更したら本ファイルも更新してください。

- 「ID」= モデル内部で使われるカラム名（英語・スネークケース）
- 「ラベル」= UI 表示用の日本語名
- 「説明」= その特徴量が何を表すか、競馬ファン向けのかみ砕き解説
- 「デフォルト」= 初期状態で選択されているか（✅ = ON）

---

## 1. 🏁 レース条件 (race_condition) — 12 features

そのレース自体の属性。距離・馬場・枠順など、馬の実力とは独立した外的条件。

| ID | ラベル | 説明 | デフォルト |
|---|---|---|---|
| `distance` | 距離 | レースの距離（m）。1200m = 短距離、2000m = 中距離、など | ✅ |
| `surface` | 馬場種別 | 芝 or ダート | ✅ |
| `track_condition` | 馬場状態 | 良 / 稍重 / 重 / 不良。前日の天候で変わる | ✅ |
| `grade` | グレード | G1 / G2 / G3 / OP / 条件戦 のレース格付け | |
| `race_class` | クラス | レースのクラス区分（1勝クラス、3勝クラス等） | |
| `field_size` | 出走頭数 | そのレースの出走馬数。頭数が少ないほど競争がゆるい傾向 | ✅ |
| `waku` | 枠番 | 1〜8番の枠。内枠/外枠で有利不利が変わる | ✅ |
| `umaban` | 馬番 | 1〜18番の馬番 | ✅ |
| `course_id` | 競馬場 | 中山・東京・阪神・京都 など | |
| `sex` | 性別 | 牡 / 牝 / セン（去勢） | |
| `age` | 馬齢 | 出走時の馬齢（2〜7歳など） | ✅ |
| `weight_carried` | 斤量 | 騎手と鞍を含む背負う重量（kg）。重いほどハンデが大きい | ✅ |
